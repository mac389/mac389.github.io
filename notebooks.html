Notebooks

This organization is an homage to Cosma Shalizi’s notebooks. These notebooks, like his, chart my attempt to distill data into knowledge and wisdom. Please use GitHub’s version control to fork and improve this.

Structure Each notebook represents a general field. The date in parentheses represents the last time I updated that notebook. I try to strike a balance between flat and linked structures. (The creation of a useful table of contents may, itself, be an instructive exercise in information management.)
